# Module Cell Registry

## Purpose

Record whole cell entries used by the KEX substrate.

## Required fields

- entry name
- family key
- lane name
- cell name
- route name
- pair name
- trace phrase
- state

## Rule

Each record describes a whole cell state.

Each record can be referenced by a workbook row or Markdown lane.

## Output

The registry should support lookup by family, lane, cell, and route.
