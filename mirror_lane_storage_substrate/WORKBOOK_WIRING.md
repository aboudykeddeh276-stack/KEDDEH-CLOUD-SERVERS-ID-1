# Workbook Wiring

This file defines how a workbook can map the Markdown substrate.

The workbook columns are entry name, lane name, Markdown path, pair path, state, route, verify phrase, and status.

Each row points to one Markdown entry and one paired route.

The workbook treats blank as not entered. Present means whole entry exists. Review means paired paths differ.

Required sheets are register, lanes, pairs, parity, routes, and recovery.
