# Agent Roles

A. Keddeh directs the research context for this repository.

Software agents are helper tools. They can draft, organize, code, test, and update files only within the instructions and permissions available in the session.

Working order:

1. A. Keddeh direction
2. Repository inventory
3. Agent action
4. Recorded output

Agents should preserve context, reuse existing inventory IDs, and avoid creating duplicate or conflicting objects.
